# Exercise 5: Test Controller POST Endpoint

## Objective
Test a POST endpoint that creates a user.

## Technologies Used
- Java 17
- Spring Boot 3.2.0
- JUnit 5
- Mockito
- MockMvc
- Maven

## Solution Test (`Exercise5_PostEndpointTest.java`)
Test the `POST /users` endpoint using `@WebMvcTest` and `MockMvc`.

## How to Run the Test
Navigate to this directory and run:
```bash
mvn test
```
