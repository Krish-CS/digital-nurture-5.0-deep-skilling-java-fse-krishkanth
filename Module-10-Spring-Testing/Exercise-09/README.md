# Exercise 9: Parameterized Test with JUnit

## Objective
Use `@ParameterizedTest` to test multiple inputs.

## Technologies Used
- Java 17
- Spring Boot 3.2.0
- JUnit 5
- Maven

## Solution Test (`Exercise9_ParameterizedTest.java`)
Test the `add` method with multiple inputs using `@ParameterizedTest`, `@CsvSource`, and `@ValueSource`.

## How to Run the Test
Navigate to this directory and run:
```bash
mvn test
```
