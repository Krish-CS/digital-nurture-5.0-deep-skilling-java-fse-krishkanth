# Exercise 6: Cursors

## Objective
Write PL/SQL blocks using explicit cursors for batch processing.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `GenerateMonthlyStatements.sql`: Generates monthly transaction statements per customer.
- `ApplyAnnualFee.sql`: Deducts annual maintenance fee from all accounts.
- `UpdateLoanInterestRates.sql`: Updates loan interest rates based on new policy rules.
