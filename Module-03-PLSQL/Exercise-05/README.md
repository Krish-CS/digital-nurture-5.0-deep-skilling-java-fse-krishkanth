# Exercise 5: Triggers

## Objective
Write triggers to automate updates and enforce business rules.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `UpdateCustomerLastModified.sql`: Auto-updates LastModified on customer record change.
- `LogTransaction.sql`: Inserts audit log records for each transaction.
- `CheckTransactionRules.sql`: Validates deposits are positive and withdrawals within balance.
