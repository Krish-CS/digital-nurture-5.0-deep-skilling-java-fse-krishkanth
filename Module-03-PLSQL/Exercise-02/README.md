# Exercise 2: Error Handling

## Objective
Write stored procedures with exception handling for bank operations.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `SafeTransferFunds.sql`: Transfers funds between accounts with rollback on error.
- `UpdateSalary.sql`: Updates employee salary with error handling for missing IDs.
- `AddNewCustomer.sql`: Inserts a customer and handles duplicate ID exceptions.
