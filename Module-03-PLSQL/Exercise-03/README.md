# Exercise 3: Stored Procedures

## Objective
Write stored procedures for processing bank operations.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `ProcessMonthlyInterest.sql`: Applies 1% monthly interest to all savings accounts.
- `UpdateEmployeeBonus.sql`: Adds a bonus percentage to salaries by department.
- `TransferFunds.sql`: Transfers funds between accounts with balance validation.
