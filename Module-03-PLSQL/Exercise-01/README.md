# Exercise 1: Control Structures

## Objective
Write PL/SQL blocks using loops and conditional logic for bank operations.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `Schema.sql`: Database schema and sample data.
- `Discount.sql`: Applies 1% interest discount to loan rates for customers above 60.
- `VIPStatus.sql`: Promotes customers with balance over $10,000 to VIP status.
- `Reminders.sql`: Sends reminders for loans due within the next 30 days.

## How to Run
1. Run Schema.sql to create tables and insert sample data.
2. Run each exercise file in order.
