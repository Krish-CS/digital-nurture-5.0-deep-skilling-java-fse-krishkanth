# Exercise 4: Functions

## Objective
Write PL/SQL functions for calculations and validations.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `CalculateAge.sql`: Function to calculate age from date of birth.
- `CalculateMonthlyInstallment.sql`: Computes the monthly EMI for a loan.
- `HasSufficientBalance.sql`: Checks if an account has enough balance.
