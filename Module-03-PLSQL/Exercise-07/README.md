# Exercise 7: Packages

## Objective
Group related procedures and functions into PL/SQL packages.

## Technologies Used
- Oracle PL/SQL

## Files Included
- `CustomerManagement.sql`: Package for adding, updating customers and getting balance.
- `EmployeeManagement.sql`: Package for hiring, updating employees and calculating salary.
- `AccountOperations.sql`: Package for opening, closing accounts and getting total balance.
