# Exercise 3: Implementing the Builder Pattern

## Objective
Develop a system to create complex objects such as a Computer with multiple optional parts using the Builder Pattern to manage the construction process.

## Technologies Used
- Java 17

## Files Included
- `Computer.java`: The product class with a private constructor and a static nested `Builder` class.
- `TestBuilder.java`: The test class demonstrating the creation of different configurations of `Computer` using the Builder pattern.
