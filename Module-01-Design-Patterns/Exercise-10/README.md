# Exercise 10: Implementing the MVC Pattern

## Objective
Develop a simple web application for managing student records using the MVC pattern.

## Technologies Used
- Java 17

## Files Included
- `Student.java`: The Model class containing attributes like name, id, and grade.
- `StudentView.java`: The View class responsible for displaying student details.
- `StudentController.java`: The Controller class handling communication between the model and the view.
- `TestMVC.java`: Main class demonstrating creating a Student, updating its details via the controller, and viewing the changes.
